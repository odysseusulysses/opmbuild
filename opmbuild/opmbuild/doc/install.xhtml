Installing OPM Builder

OPM Builder is a Java library. A JAR file containing the current stable release can be found here:

opmbuild.jar

Generating Graphics

One optional feature of OPM Builder is the facility for generating graphical depictions of OPM graphs using GraphViz. If you wish to use this feature, you need to install GraphViz.
